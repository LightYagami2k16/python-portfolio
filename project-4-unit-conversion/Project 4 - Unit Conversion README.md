# Unit Converter (Python)

## 📖 About
This is a simple Python program that converts values between common units:
- Kilometers → Miles
- Celsius → Fahrenheit
- Kilograms → Pounds

It uses basic `if-else` statements to let the user choose which conversion to perform.

## ▶️ How to Run
1. Clone this repository or download the file.
2. Run the script in your terminal:
   ```bash
   python conversion.py

Choose a conversion:
1. Kilometers to Miles
2. Celsius to Fahrenheit
3. Kilograms to Pounds
Enter your choice (1/2/3): 1
Enter distance in kilometers: 10
