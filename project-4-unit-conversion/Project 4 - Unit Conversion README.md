# Unit Converter
A basic Python program that converts values between kilometers and miles, Celsius and Fahrenheit, and kilograms and pounds.

## ⚙️ What It Does
- Prompts the user to choose a conversion type.
- Performs the calculation using simple formulas.
- Displays the converted value with clear output.

## ▶️ How to Run
1. Clone or download this repository.
2. Navigate to the `project-4-unit-converter` folder.
3. Run the script:
