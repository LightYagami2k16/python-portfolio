# Basic Unit Converter

## 📖 Short Description

**Basic Unit Converter** is a simple Python command-line application that converts values between common units of measurement. It provides an easy way to perform unit conversions without manually calculating conversion formulas.

---

## ✨ What It Does

The program allows users to:

* Convert **length** (e.g., meters to kilometers, centimeters to meters)
* Convert **weight** (e.g., kilograms to grams, pounds to kilograms)
* Convert **temperature** (e.g., Celsius to Fahrenheit and vice versa)
* Accept user input through the keyboard
* Display the converted result in a clear and readable format
* Handle invalid menu choices and input errors gracefully

---

## ▶️ How to Run It

### Prerequisites

* Python 3.x installed on your computer

### Steps

1. Open a terminal or command prompt.
2. Navigate to the folder containing the program.

```bash
cd path/to/your/project
```

3. Run the program.

```bash
python unit_converter.py
```

*(Replace `unit_converter.py` with your actual Python filename if it is different.)*

4. Follow the on-screen instructions:

   * Choose the type of conversion.
   * Enter the value to convert.
   * View the converted result.

## 🐍 Python Concepts Used

This project demonstrates several fundamental Python programming concepts:

* **Variables** – Store user input and conversion values.
* **Data Types** – Work with integers and floating-point numbers.
* **Input and Output** – Use `input()` and `print()` for user interaction.
* **Conditional Statements (`if`, `elif`, `else`)** – Execute different conversions based on user selections.
* **Loops (`while`)** – Allow the program to continue running until the user chooses to exit.
* **Functions** – Organize conversion logic into reusable blocks of code.
* **Arithmetic Operators** – Perform mathematical conversion calculations.
* **Error Handling (`try` and `except`)** – Prevent crashes from invalid numeric input.
* **Menu-Driven Programming** – Create an interactive command-line interface.
